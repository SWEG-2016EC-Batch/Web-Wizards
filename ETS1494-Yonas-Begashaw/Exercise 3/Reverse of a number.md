# Problem Analysis   
**Input:** A number.  
**Output:** Reverse of the number.

# Operation:   
- Variable declaration that holds an input and operation results.
- Input number.
- Calculate digit: digit = number % 10.
- Display digit.
- Update number: number = number / 10.
- If number is not equal to zero, go to step 3.
