**Input:** row.
**Output:** Half pyramid with numbers.

- Variable declaration that holds an input and operation results.
- Input row and column.
- Initialize variable i to 1.
- Iniitialize variable j to 1.
- Display j.
- Update j: j = j+1.
- If j <= i, go to step 5.
- End the line.
- Update i: i = i + 1.
- If i <= row, go to step 4.
