# Problem Analysis     
**Input:** rows.
**Output:** Inverted half pyramid.

# Operation:   
- Variable declaration that holds an input and operation results.
- Input number of rows.
- Initialize variable i to row.
- Iniitialize variable j to 1.
- Display '*'.
- Update j: j = j + 1.
- If j <= i, go to step 5.
- End the line.
- Update i: i = i - 1.
- If i > 0, go to step 4.
